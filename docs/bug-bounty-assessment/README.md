This folder implements the **External bug bounty assessment plan**: repeatable steps from policy intake through reporting. It does **not** replace reading the live program policy on HackerOne/Bugcrowd/etc.

| Step | Document |

| 1 | [01-policy-intake.md](01-policy-intake.md) — scope, rules, automation, disclosures |

The ReconX Elite product in this repository is a **recon automation platform**. These docs are **operational templates** for your external engagements unless a program explicitly lists this codebase as in scope.
